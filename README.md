# Bienvenido a mi prueba técnica 👋
Panel de administración de productos y usuarios

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npm run dev
   ```

## Stack

   1. ReactJS v19.1.10
   2. CSS frameworks: MUI , Tailwind
   3. Router: ReactRouter v7.8.2
   4. Bundling: Vite v7.1.2
   5. Gestión de estados: Zustand v5
   6. Promise API: Axios
